# تقرير دعم اللغات واللهجات في النظام

## اللغات المدعومة: 15 لغة و 4 لهجات عربية

### 1. اللغات الرئيسية المفعلة

#### اللغات العربية (4 لهجات):
- **السعودية** (`ar-SA`): اللهجة السعودية الرسمية
- **الإماراتية** (`ar-AE`): اللهجة الإماراتية 
- **المصرية** (`ar-EG`): اللهجة المصرية العامية
- **العربية الفصحى** (`ar`): العربية الكلاسيكية

#### اللغات الأوروبية (6 لغات):
- **الإنجليزية الأمريكية** (`en-US`)
- **الإنجليزية البريطانية** (`en-GB`)
- **الفرنسية** (`fr-FR`)
- **الألمانية** (`de-DE`)
- **الإسبانية** (`es-ES`)
- **الإيطالية** (`it-IT`)

#### اللغات الآسيوية (3 لغات):
- **اليابانية** (`ja-JP`)
- **الصينية** (`zh-CN`)
- **الهندية** (`hi-IN`)

#### اللغات الأخرى (2 لغة):
- **الروسية** (`ru-RU`)
- **البرتغالية** (`pt-BR`)

## 2. أماكن التفعيل في النظام

### أ. نظام المكالمات الصوتية (Voice Webhook)
**الموقع**: `server/voice-webhook.ts` و `server/routes.ts`
**الوظيفة**: تحديد اللغة تلقائياً حسب رقم الهاتف

```typescript
const languageMap = {
  '+966': { language: 'ar', speechLang: 'ar-SA', culture: 'saudi' },
  '+971': { language: 'ar', speechLang: 'ar-AE', culture: 'emirati' },

  '+1': { language: 'en-US', speechLang: 'en-US', culture: 'american' },
```

```typescript
}
```

### ج. واجهة المستخدم (Frontend)
**الموقع**: `client/src/index.css`
**الوظيفة**: دعم RTL للعربية وخطوط متعددة

```css
font-family: 'IBM Plex Sans Arabic', 'Inter', system-ui, sans-serif;
direction: rtl; /* للعربية */
```

### د. نظام التصدير والتقارير
**الموقع**: `server/export-service.ts`
**الوظيفة**: تواريخ وتنسيقات محلية


## 3. التحديد التلقائي للغة

### أ. حسب رقم الهاتف:
- `+966` → السعودية (ar-SA)
- `+971` → الإمارات (ar-AE)  
- `+20` → مصر (ar-EG)
- `+1` → أمريكا (en-US)
- `+44` → بريطانيا (en-GB)

### ب. حسب السياق:
- المكالمات الصوتية: تحديد تلقائي
- واجهة الويب: عربية افتراضية
- التقارير: حسب إعدادات المستخدم
